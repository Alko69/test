# Commandes Git
## Première initialisation
```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/jperaudon/validation_html.git
git push -u origin main
```

## Mises à jour suivantes
```bash
git add .
git commit -m "nice message"
git push -u origin main
```
